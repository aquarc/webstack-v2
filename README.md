# webstack v2

This project contains the backend and the frontend for aquarc!

## git clone

```bash
git clone https://github.com/aquarc/webstack-v2.git
cd webstack-v2
```

## build the frontend

```bash
cd frontend
npm run build
cd ..
```

## build the backend

```bash
go build
```

You will have to install go_sqlite3 properly

## run the code

On windows:

```bash
serve.exe
```

On everything else:

```bash
./serve
```
